## 🎬 Movie Explorer App

**Movie Explorer** is a React-based web application that allows users to search and explore movies using the **OMDb API**. The app fetches movie data from an external API and displays it in a simple and user-friendly interface.

On the first load, the application shows different movies automatically. Users can search for any movie by typing its name in the search bar. The app then fetches matching results from the API and displays them instantly.

Each movie card includes a **Details** button. When the user clicks this button, the application navigates to a separate page using routing where detailed information about the selected movie is displayed.

This project helped me practice working with APIs, React components, and routing while building a clean UI.

### ⚙️ Features

* Browse movies on the homepage
* Search movies by name
* View detailed movie information
* Dynamic API data fetching
* Client-side routing

### 🛠 Tech Stack

* HTML
* CSS
* JavaScript
* React

### 🌐 API

* **OMDb API**
